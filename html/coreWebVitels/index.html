->

->

If user is viewing a page the whole page will be loaded but the biggest element which is important is loaded in how much time. (2.5sec in the the page will load is best as google)

FID(First Input Delay)

->

If user click on any input element So how long does that input process.(According to google web dev input should start processing in less than 100 milliseconds).
